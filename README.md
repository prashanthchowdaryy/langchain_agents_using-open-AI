# AI Data Analyst using LangChain and OpenAI

## Overview

This project is an AI-powered data analysis assistant built using LangChain, OpenAI, and Pandas. It allows users to interact with datasets using natural language queries instead of writing code.

The system uses a language model to understand user questions and dynamically execute Python code to generate answers based on the dataset.
---
## Features
- Query datasets using natural language
- Automatic data analysis using a language model
- Support for multiple DataFrames
- Handles missing data (basic preprocessing included)
- Interactive question-answering loop
- Works with CSV datasets

--
## Technologies Used
- Python
- LangChain
- OpenAI API
- Pandas
- Jupyter Notebook
---
## Dataset
The project uses the Titanic dataset for demonstration purposes. You can replace it with any CSV file.
